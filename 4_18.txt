##### Java

■アジェンダ
・基本データ型と参照型（スタック・ヒープ）
・配列
・文字列どうしの比較（equalsメソッド）
・switch文
・for文, 拡張for文

◎List
　⇒可変長で複数の要素を格納できる
　・List.size()
　　⇒要素数の取得（array.lengthのイメージ）
　・List.get(i)
　　⇒要素の取得（array[i]のイメージ）

◎スタックとヒープ
　・int, float, double（プリミティブ型） ⇒ スタック領域で管理
　・Class, 配列（newで宣言するもの） ⇒ ヒープ領域で管理
　※Stringは参照型なので注意
　　⇒実はnewが使われているが省略されている
　　ex) String a = new String("りんご");
　　    String b = new String(new char[]{'り', 'ん', 'ご'});
　　    aもbも「りんご」という出力になる
　※上の例で「a == b」とすると参照先アドレスの比較になってしまうためtrueにならない
　※配列で「a == b」のようにするとアドレスが一致してしまい、
　　片方を書き換えると参照先配列が書き変わってしまうため注意

◎Arrays.asList()
　⇒配列をListに変換する
　※このメソッドで作成したリストは、リストでありながら固定長配列となるため、
　　新しく要素を追加したり削除したりできない（addできない）
